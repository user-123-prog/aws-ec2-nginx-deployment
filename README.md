# AWS EC2 Nginx Deployment Project

## Project Overview
This project demonstrates how to deploy a website on an AWS EC2 instance using Nginx.

## Tools Used
- AWS EC2
- Ubuntu Linux
- Nginx Web Server

## Steps Performed
1. Launched an EC2 instance
2. Connected to server using SSH
3. Installed Nginx
4. Created a simple HTML webpage
5. Hosted website using public IP

## Commands Used

sudo apt update
sudo apt install nginx -y
sudo systemctl start nginx

## Output
Website successfully hosted on AWS EC2 instance.
